Algorithm Explanation

Linear Search checks each element in the array one by one from the start until it finds the target value or reaches the end of the array. It works on both sorted and unsorted arrays but has a time complexity of O(n) in the worst case.

Binary Search repeatedly divides the sorted array in half, checking if the target value is in the left or right half until the target is found or the search space is exhausted. This has a time complexity of O(log n) but requires the array to be sorted.

Jump Search works on sorted arrays by jumping ahead by fixed steps until it finds an element greater than the target, then performs a linear search in the previous block. Optimal step size is √n, giving a time complexity of O(√n).

Interpolation Search is an improved binary search for uniformly distributed sorted arrays. It estimates the position of the target based on the key values, achieving average case complexity of O(log log n) but O(n) in worst case.

Exponential Search finds the range where the target might be by doubling the index until finding a range that contains the target, then performs binary search within that range. Time complexity is O(log n) and works well for unbounded searches.

function linearSearch(array, target):
    for index from 0 to length(array) - 1:
        if array[index] == target:
            return index  // Element found
    return -1  // Element not found
                
function binarySearch(array, target):
    left = 0
    right = length(array) - 1

    while left <= right:
        mid = floor((left + right) / 2)
        if array[mid] < target:
            left = mid + 1
        else if array[mid] > target:
            right = mid - 1
        else:
            return mid  // Target found
    return -1  // Target not found
                
function jumpSearch(array, target):
    n = length(array)
    step = floor(√n)
    prev = 0

    // Jump ahead in steps
    while array[min(step, n)-1] < target:
        prev = step
        step += floor(√n)
        if prev >= n:
            return -1

    // Linear search in the block
    while array[prev] < target:
        prev++
        if prev == min(step, n):
            return -1

    if array[prev] == target:
        return prev
    return -1
                
function interpolationSearch(array, target):
    low = 0
    high = length(array) - 1

    while low <= high and target >= array[low] and target <= array[high]:
        // Estimate position using interpolation formula
        pos = low + floor(((target - array[low]) * (high - low)) / (array[high] - array[low]))

        if array[pos] == target:
            return pos
        if array[pos] < target:
            low = pos + 1
        else:
            high = pos - 1

    return -1
                
function exponentialSearch(array, target):
    // If target is at first position
    if array[0] == target:
        return 0

    // Find range for binary search by doubling index
    i = 1
    while i < length(array) and array[i] <= target:
        i = i * 2

    // Perform binary search in the found range
    return binarySearch(array, target, i/2, min(i, length(array)-1))
